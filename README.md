# カード精算アプリ

三井住友系カードのCSVを読み込み、夫婦間の精算額を自動計算するWebアプリです。

## 機能
- CSVアップロード（三井住友系フォーマット対応）
- 夫婦の負担割合をスライダーで設定
- 夫・妻それぞれの固定費（ローンなど）を登録
- 明細テーブルから按分対象外の項目を選択
- 夫→妻（または妻→夫）の精算額を自動計算

## 計算式
```
夫の総負担 = (夫固定費 + 妻固定費) ÷ 2 + カード合計(対象分) × 夫割合
夫→妻 支払額 = 夫の総負担 − 夫固定費
```

## ローカル開発

```bash
npm install
npm run dev
```

## デプロイ（Vercel）

1. このリポジトリをGitHubにプッシュ
2. [Vercel](https://vercel.com) でリポジトリをインポート
3. Framework Preset: **Vite** を選択
4. そのままデプロイ
